# 更新日志

本文件记录项目的主要变更，按日期倒序排列。

## 2026-07-23

### 新增
- **卡组详情预览支持本地卡牌图片**：推荐卡组弹窗（DeckDetailModal）按旅法师营地风格重写，左侧卡组列表展示费用宝石、类型图标、稀有度着色牌名（白/黑、蓝、紫、橙）、缩略图与数量，点击卡牌在右侧切换完整大图。
- **卡牌图片资源改用 Git LFS 管理**：新增 `.gitattributes` 用 LFS 跟踪 `public/hearthstone-cards/**`（742 缩略图 + 744 大图 + 1 个 `cards_meta.json`，约 93MB），并调整 `.gitignore` 去除对该目录的忽略，使图片随仓库发布、线上不再 404。
- **新增工具与脚本**：
  - `src/hearthstone-achievements/utils/cardImages.js`：本地图查询、稀有度标准化、颜色/类名，暴露 `CDN_BASE` 用于缺失图回退。
  - `src/hearthstone-achievements/data/deck-card-images.json`：卡牌名称到本地 crop/full 路径映射。
  - `scripts/find-deck-cards.mjs` / `copy-deck-images.mjs` / `generate-deck-card-manifest.mjs`：统计、复制、生成卡牌图片清单。
- **卡组弹窗跟随主页主题**：背景/文字/边框改用与主页同一套 `--hs-*` 主题变量，亮色主题弹窗为亮色（白卡文字黑色）、暗色主题为暗色（白卡文字白色）。

### 修复
- **白卡文字不可见**：弹窗深色背景下原白卡文字为黑色，已改为跟随主题（亮色黑、暗色白）。
- **5 个成就职业归属错误**（职业名已按项目规范用全称修正）：
  - 语声鼎沸 → 法师
  - 污手马仔 → 圣骑士
  - 法力洼地 → 萨满祭司
  - 食量极限 → 恶魔猎手
  - 都缝好了 → 死亡骑士

### 优化
- **顶部吸顶重组**：「我的成就」子切换（按版本/按职业/待完成清单）+ 版本/职业选择 + 操作行并入同一吸顶块，滚动时整体固定；版本名/职业名改为金色胶囊芯片显示在子切换左侧。
- **展开/收起全部位置统一**：从悬浮按钮移至版本/职业选择之后同一行，在「按版本浏览 / 我的-按版本 / 我的-按职业」三处位置与触发条件一致。
- **图片兜底**：弹窗缩略图缺失时回退官方 CDN，避免破图。
